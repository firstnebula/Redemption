Welcome to WoW-Mania Redemption Realm (Bug Tracker)

READ CAREFULLY:

Before entering any issue/bug please search the repository and make sure it was not already reported by someone else. If it was do not create a new one, just comment and confirm the existing one adding any missing detail.

Please use the following tags for bug titles:

[Quest] - Any quest related issue [NPC] - Any mob or NPC issue [Item] - Any item or gameobject issue [Map] - Any map issue [Spell] - Any spell, ability or skill [Core] - Any core issue (code) [Achievement] - Any achievement related issue When entering an issue please follow this procedure:

Give name and id to the related object/mob/quest (check wowhead.com) Described the issue. What is actually happening (actual behaviour). Describe what you think it should be happening (expected behaviour). Additional information (i.e.: is it a quest chain, is it a class specific)
